Etiqueta Head Elementos en bloque Elementos en línea

samp

Indica que el texto delimitado por la etiqueta es un mensaje proporcionado por un ordenador. Se suele utilizar de manera conjunta con el elemento <pre> para que se respeten espacios, tabuladores y saltos de línea. 

            
                <p>
                <pre><samp>
                Ha introducido una dirección incorrecta.
                Pulse F10 para reiniciar el proceso. 
                </samp></pre>
                <p>Descatalogado</p>

Si introduce la dirección de manera incorrecta, el sistema mostrará la siguiente información: 


              Ha introducido una dirección incorrecta.
              Pulse F10 para reiniciar el proceso.
