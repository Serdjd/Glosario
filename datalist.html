datalist

Define un conjunto de elementos <option> como una lista para su uso en como una listas de formulario. Habitualmente en elementos input de tipo text, aparece como una lista de opciones al pulsar sobre el componente o genera una ayuda contextual filtrando las opciones según se escriben los caracteres en la caja de texto. 

            
                <input type="text" list="lenguajes" placeholder="Selecciona un lenguaje">
                <datalist id="lenguajes">
                <option>Java</option>
                <option>C#</option>
                <option>Cobol</option>
                <option>Pascal</option>
                <option>Python</option>
                </datalist>
