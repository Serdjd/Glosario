template

Permite crear fragmentos de documento (componentes) que pueden ser utilizados como plantillas. Es necesario utilizar JavaScript para rellenar estas plantillas.
Es una forma de mantener el contenido HTML en el lado del cliente que no se renderiza cuando se carga una página, pero puede ser instanciado posteriormente durante el tiempo de ejecución usando JavaScript. 

            
                <template id="myTemplate"> 
                <p>Hola, soy un párrafo en un template.</p>
                </template>
