span

El uso más habitual de este elemento consiste en delimitar una parte de un texto para asignarle un estilo distinto al del resto sin generar un nuevo bloque. Este elemento se utiliza cuando no existe otro elemento de línea más apropiado, por ejemplo <strong> o <em>.
No tiene ningún efecto en el contenido o el diseño hasta que se aplica estilo CSS o se manipula con JavaScript 

            
                <p> 
                Una olla de algo más vaca que carnero, salpicón las más noches, duelos y
                quebrantos los <span class="diassemana">sábados</span>, lantejas los
                <span class="diassemana">viernes</span>, algún palomino de añadidura los
                <span class="diassemana">domingos</span>, consumían las tres partes de su hacienda.
                </p>
