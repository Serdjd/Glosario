select

Permite al usuario elegir una opción de un conjunto de ítems mostrado como una lista de opciones (puede mostrarse como lista desplegable). Las opciones se establecen con el componente <option> y se pueden agrupar con el componente <optgroup>.
Atributos significativos:

  • multiple: indica si admite selección múltiple.
  • size: indica el número de líneas que se mostrarán simultáneamente.

            
                <form>
                <select>
                <option>Baloncesto</option>
                <option>Fútbol</option>
                </select>
                </form>
