script

Permite embeber código ejecutable en el documento HTML. Estos programas son interpretados y ejecutados por el navegador, por lo que no suponen una carga de trabajo para el servidor
Atributos significativos:

  • src: contiene la URI del programa.
  • type: indica el lenguaje de programación utilizado.
            
                <script>
                function sumar (a, b) {
                    var resultado = a + b;
                    return resultado;
                }
                </script>
