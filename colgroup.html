colgroup

Permite crear grupos de una o más columnas. También permite asignar atributos comunes a todas las columnas agrupadas. La agrupación se realiza de izquierda a derecha de manera consecutiva utilizando el atributo span o agregando elementos <col>.
Atributos significativos:

  • span: determina el número de columnas consecutivas agrupadas por el elemento <colgroup>. Este número debe ser un entero positivo. 

            
                <colgroup span="2" style="background-color: bisque;"></colgroup>
