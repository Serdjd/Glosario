Etiqueta Head Elementos en bloque Elementos en línea

label

Identifica una etiqueta de un campo de entrada de datos del formulario. También permite poner el foco en el campo de entrada pulsando sobre el texto de la etiqueta.
La asociación se puede realizar de dos maneras: conteniendo el campo de entrada entre las etiquetas de apertura y cierre del elemento <label> o utilizando el atributo for cuyo valor deberá ser el del atributo id del componente del formulario relacionado.
Atributos significativos:

  • for: contiene la referencia del id del componente con el que está asociado

            
                <p>
                <label for="apodo">Apodo</label>
                <input type="text" id="apodo">
                </p>
